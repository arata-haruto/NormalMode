「癒やしエフェクトアニメ素材セット」

●収録内容

2020年4月に支援サイト限定素材として公開した「癒やし・回復のエフェクトアニメ素材」から一部をぴぽやサイト向けに配布するものになります

アニメパターンをスプライトシート状に並べたPNG画像となっています

アニメーションの表示速度は15fpsで作成されています

ファイルサイズの最適化はしておりませんので、必要に応じて下記の「注意事項」で紹介しているアプリなどでファイルサイズを小さくすることをお奨めします
